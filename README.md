# StudioVault : Tattoo & Piercing Booking Platform

Human-centered booking, consent, and compliance system
  for tattoo and piercing studios.

## Principle
- Consent without evidence is not consent
- Compliance is enforced inthe system
- Artists retain ownership of their work and clients
- Client data is minimized by design

## Stack
- MongoDB
- Express
- React
- Node.js

## Status
Scaffold only.  No domain logic implemented yet.
